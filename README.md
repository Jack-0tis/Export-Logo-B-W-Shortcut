# Export-Logo-B-W-Shortcut
Illustrator Script that saves a Black and White version of the file

This script will save a 0,0,0,100 (black) and a 0,0,0,0 (white)version of a logo as an .ai file, it will save it in the same directory that the colour logo is saved. If the colour logo is not saved in any directory it will ask the user to choose a folder to save it in.
